# RHCSA Learning Journey

Repositorio personal donde documento mi preparación para RHCSA.
Incluye apuntes teóricos, prácticas de laboratorio, ejercicios resueltos y evidencia de progreso.

## Objetivos
- Aprender administración Linux de forma práctica
- Prepararme para RHCSA
- Documentar mi progreso en GitHub
- Construir un portafolio técnico

## Contenido
- `notes/`: apuntes por capítulo o tema
- `labs/`: prácticas documentadas
- `scripts/`: scripts usados durante el aprendizaje
- `progress/`: seguimiento de avance y checklist

## Temas estudiados
- Comandos básicos
- Gestión de usuarios y grupos
- Permisos
- Procesos y servicios
- Redes
- Almacenamiento
- Systemd
